# homeapp data create


* **Read_data.ipynb**

  讀取mobile01的文章，並擷取出現產品、廠牌的段落。裡面有滿多統計，所以程式碼很多。但如果老師沒有要你修改原始資料，這邊就不用看。 

* **Labelapp.ipynb**

  計算兩組標記人員的kappa、統計、依照不同任務產生不同形式的資料。總之程式碼超級長。 
  
* **labeled_data_origin**
  兩組人馬標記完成資料的原始檔(.json) 
  
  **label1.json** : 標記人員1的標記結果
  
  **label2.json** : 標記人員2的標記結果
  
* **labeled_data_origin**

  資料標記程式(網頁)
